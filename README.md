# Scaffold : Simple Front

A simple scaffold project to quick boot a simple front POC.

## Stack

* webpack
* eslint
* babel
* sass
* stylelint

## Included

* dotenv
* a simple css reset

# Usage

## Install dependencies

```
npm install
```

## Configure your local env

```
cp .env.dist .env
vi .env
```

## Run dev server

```
npm run start
```

## Build for production

```
npm run build
```
